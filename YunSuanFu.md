### 各类运算符

- 算数运算符： +  -  *  /  %  ++  --  

- 关系运算符： ==    !=    >    <    >=    <=

- 位运算符：  &  |  ^  ~  <<  >>  >>>

- 逻辑运算符：&&  ||  !

- 赋值运算符:  =  +=  -=  *=  /=  (%)=  <<=  >>=  &=  ^=  !=

- 条件运算符:    ?   :

- instanceof运算符：检查该对象是否是一个特定类型



#### 优先级关系

| 类别     | 操作符                                          | 关联性   |
| :------- | ----------------------------------------------- | -------- |
| 后缀     | （）[] .(点操作符)                              | 从左到右 |
| 一元     | expr++  expr--                                  | 从右到左 |
| 一元     | ++expr  --expr+ - ~ !                           | 左到右   |
| 乘性     | * / %                                           | 左到右   |
| 加性     | + -                                             | 左到右   |
| 移位     | >>  >>>  <<                                     | 左到右   |
| 关系     | >  >=  <  <<                                    | 左到右   |
| 相等     | ==  !=                                          | 左到右   |
| 按位与   | &                                               | 左到右   |
| 按位异或 | ^                                               | 左到右   |
| 按位或   | \|                                              | 左到右   |
| 逻辑与   | &&                                              | 左到右   |
| 逻辑或   | \|\|                                            | 左到右   |
| 条件     | ？  ：                                          | 从右到左 |
| 赋值     | =  +=  -=  *=  /=  %=  >  >=  <  <=  &=  ^= \|= | 从右到左 |
| 逗号     | ,                                               | 左到右   |
